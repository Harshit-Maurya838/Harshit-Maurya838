## About Me

Specializing in distributed systems, real-time systems, and AI applications. Systems engineering with a preference for building scalable and correct systems with performance being an absolute constraint from the very beginning.

I operate across the stack, from low-level networking and async systems to AI-driven applications and developer tooling. My work is driven by a bias toward execution: shipping systems that handle real users, real load, and real complexity.

Currently building:

- *AI-powered systems* (agent workflows, evaluation pipelines, applied ML)
- *Real-time applications* (Socket.io, LAN-based P2P systems, collaboration tools)
- *Scalable backend architectures* (Next.js, distributed APIs, RBAC systems)
- *Developer-focused tools* (automation, productivity systems, infra abstractions)
